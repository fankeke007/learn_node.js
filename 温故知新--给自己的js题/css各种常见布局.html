居中

  • 水平居中

    • 行内元素:text-align:center
    • 定宽块状元素:margin:x auto;(即设定左右margin为auto)
    • 不定宽块状元素：设置子元素为display:inline;父元素设置：text-align:center;
    • 通用方案（移动端，pc端ie9+）:父元素设置display:flex;justify-content:conter;

  • 垂直居中

    • 父元素一定，子元素为单行内联文本：设置父元素的height等于行高line-height
    • 父元素一定，子元素为多行内联文本：设置父元素的display:table-cell或inline-block，再设置vertical-align:middle;
    • 块状元素:设置子元素position:absolute;并设置top、bottom为0，父元素要设置定位为static以外的值，margin:auto;
    • 通用方案: flex布局，给父元素设置{display:flex; align-items:center;}。

			border: 1px solid #ccc ;
			width: 600px;
			height: 100px;
		}
		.v_center_child_1{
			line-height: 100px;
		}
我是单行内联文本,我要垂直居中
