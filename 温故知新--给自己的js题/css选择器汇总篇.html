css选择器汇总篇

基本选择器

  • 1.元素选择器：elementname(元素名称)
  • 2.类选择器：.classname(类名)
  • 3.ID选择器：#idname("标志名")
  • 4.通配选择器：*
  • 5.属性选择器：[attr=value]

组合选择器

  • 1.相邻兄弟选择器：A + B
  • 2.普通兄弟选择器：A ~ B

    在使用 ~ 连接两个元素时,它会匹配第二个元素,条件是它必须跟(不一定是紧跟)在第一个元素之后,且他们都有一个共同的父元素 .

  • 3.子选择器：A > B
  • 4.后代选择器： A B

普通兄弟选择器 A~B

			.nb~span{
				color: green;
			}
